About Me

Hi, I'm Xander Rancap — a software developer studying Software Development at SAIT with a 3.7 GPA. I combine technical expertise with strong teamwork and communication skills built through years in customer-focused roles.

In 2024, I interned at Eduspec Holdings Berhad, working hands-on with Arduino, SAM Labs, and VEX Robotics. I contributed to circuit design and programming tasks, gaining practical hardware-software experience while strengthening leadership and collaboration.

I’ve also led several projects applying object-oriented programming principles, including a cross-platform app built with .NET MAUI Blazor Hybrid. These experiences deepened my understanding of scalable software design and project leadership.

My toolkit includes C#, Python, JavaScript, SQL, HTML, and CSS, with UI design skills in Figma. I’m passionate about creating clean, user-centered solutions and eager to keep growing and collaborating in the tech space.

Projects

Gym Web Application

.Net Maui Blazor-Hybrid

This project is a cross-platform gym management web application built with .NET MAUI Blazor Hybrid in C#. It combines the performance of native applications with the flexibility of web technologies, allowing it to run smoothly on both desktop and mobile devices. The app features a secure login and registration system, a clean landing page with intuitive navigation, and a fully integrated database that manages users, memberships, schedules, and other gym-related data.

In addition to core functionality such as profile management and membership tracking, the design emphasizes responsiveness and usability, ensuring a seamless experience across platforms. This project highlights my skills in full-stack development, database integration, and building hybrid applications that are scalable, modern, and user-friendly.

NV Closet

Figma Mock-up

NV is a modern closet management application designed to help users organize their wardrobe and generate personalized outfit suggestions. Built with a clean and intuitive interface, the app allows users to digitally store and categorize their clothing items, making it easy to browse and plan outfits. A key feature of NV is its AI integration, which analyzes the user’s stored clothes and automatically generates stylish outfit combinations based on season, occasion, or personal preferences.

With its focus on both functionality and design, NV shows my skills in UI/UX design. The project shows how technology can simplify everyday routines by combining smart recommendations with a user-friendly experience.
